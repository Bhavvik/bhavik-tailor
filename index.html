Project Title: E-commerce Platform

This project is a feature-rich e-commerce platform built with modern web technologies including React.js for frontend, Node.js for backend APIs, and MongoDB as the database. It supports user authentication, product browsing, cart management, checkout flow, admin panel, order tracking, and payment integration.

Key Features

  • User registration and login with JWT
  • Role-based access control for admin
  • Responsive product catalog and filtering
  • Shopping cart and order checkout
  • Payment gateway integration (Stripe)
  • Order management dashboard
  • Dark and Light mode support

Technologies Used

React.js Node.js Express MongoDB JWT Tailwind CSS Stripe
